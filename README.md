# 介绍

项目是基于Vue3 、TypeScript 、Vite 、Element Plus 、Pinia 等组件的后台管理系统

# 代码分支

* dev
* test
* pro

# TS代码规范

* 编辑器安装好ESlint和Prettier插件/扩展。
* ESlint配置后，如果编写代码时不符合规则规范，页面将会出现红色波浪线或黄色波浪线。
* 执行npx prettier --write .命令，可将代码的引号，分号，换行，缩进格式化为设定的规范。

# 开发
安装依赖:

```shell
pnpm install
```

运行:

```shell
pnpm run dev
```
